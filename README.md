## 🚀 Texnologiyalar

- **Frontend:** React, Axios, React Router
- **Backend:** Node.js, Express.js, JWT, MongoDB
- **UI:** TailwindCSS yoki Bootstrap (tanlovga qarab)
- **Autentifikatsiya:** JWT va bcrypt
